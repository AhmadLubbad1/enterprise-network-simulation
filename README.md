# 🌐 Enterprise Network Infrastructure Simulation

A simulated enterprise-level network designed and built using GNS3 
with VMware integration for realistic server virtualization.

## 🗺️ Network Design
- OSPF dynamic routing for traffic optimization and path redundancy
- VLAN segmentation for network isolation and security
- IPv4 addressing and subnetting across multiple segments
- End-to-end connectivity testing across all client-server nodes

## 🖥️ Servers Deployed
- DNS — domain name resolution
- FTP — file transfer service
- SMTP — email communication
- VoIP — voice over IP communication

## 🛠️ Tools Used
- GNS3
- VMware
## 📄 Files
- `configs.md` — full router CLI configurations for all 4 routers
